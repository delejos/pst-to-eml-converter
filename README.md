# PST → EML Converter (Free & Open Source)

A free Windows tool that converts Outlook PST files to EML format.
Works fully offline using Microsoft Outlook.

## Features
- Convert single PST or folders with PST files
- Preserve folder structure
- Export EML files
- Best-effort attachment export
- Progress + detailed logs
- No internet connection required

## Requirements
- Windows
- Microsoft Outlook installed

## Download
👉 Download the MSI from the Releases page.

## Known limitations
- Non-mail items (Contacts, Calendar, Tasks) are skipped
- Some Outlook attachments may be skipped if Outlook blocks them
